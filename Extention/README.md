# Homescreen Schedule Extension

このプロジェクトは、時間割とテスト日程を管理できるブラウザの「新しいタブ」画面の拡張機能です。

## 📁 ファイル構造と役割

- **`manifest.json`**
  拡張機能の基本設定ファイルです。拡張機能名や、`index.html`を新しいタブとして読み込むための設定が書かれています。

- **`index.html`**
  画面の骨組み（HTML）です。ここに文字の構造やボタンの配置が定義されています。文字を変えたい場合や、要素を追加したい場合はここを編集します。

- **`style.css`**
  画面の見た目（デザインやレイアウト、色など）を制御するファイルです。
  - **色やテーマを変えたい場合**：上部にある `:root { ... }` の部分にある変数を編集してください。
  - **各パーツの大きさを変えたい場合**：対応するクラス（`.timetable-wrapper` や `.calendar-grid-wrapper` など）の `width` や `height` などを編集します。

- **`script.js`**
  画面の動きやデータ保存（JavaScript）を制御するファイルです。
  - 時計の更新、タブの切り替え、時間割のデータ保存、ブックマークの動作などはすべてここに書かれています。
  - **デフォルトの時間割構造を変えたい場合**：ファイル上部にある `schedule` や `periods`、`dayNames` などのデータを書き換えることで、授業時間や曜日、デフォルトの科目を変更できます。

---

## 🛠 カスタマイズ方法の例

### 1. 時間割のデフォルトデータを変更する
`script.js` の最初の方にある `schedule` 配列を書き換えます。
```javascript
  const schedule = [
    ['I-a','II-a','III-a','IV-a','V-a','VI','LHR'], // 月曜日
    // ...
  ];
```
また、`periods` に授業の開始・終了時間が入っています。

### 2. テーマカラー（色）を変更する
`style.css` の先頭にある変数を変更します。
```css
  :root {
    --bg-primary: #0f0f1a; /* 一番後ろの背景色 */
    --color-I: #6ec6ff;    /* グループIの色 */
    // ...
  }
```

## 🚀 拡張機能のインストール方法

### Google Chrome / Microsoft Edge の場合
1. ブラウザのアドレスバーに `chrome://extensions/`（Edgeの場合は `edge://extensions/`）と入力して拡張機能管理ページを開きます。
2. 右上にある **「デベロッパーモード」** のスイッチをオンにします。
3. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
4. このプロジェクトのフォルダ（`index.html`などが入っているフォルダ）を選択します。
5. 新しいタブを開くと、この画面が表示されます。（※初回は「変更を維持しますか？」と聞かれる場合があるので「維持する」を選択してください）

### Mozilla Firefox の場合
1. ブラウザのアドレスバーに `about:debugging#/runtime/this-firefox` と入力してページを開きます。
2. **「一時的な拡張機能を読み込む」** をクリックします。
3. フォルダ内の `manifest.json` ファイルを選択します。
4. 新しいタブを開くと、この画面が表示されます。（※Firefoxでは一時的な読み込みのため、ブラウザを再起動するたびに再度読み込む必要があります。恒久的に使用したい場合は、`about:config`の設定変更などが必要になります）
